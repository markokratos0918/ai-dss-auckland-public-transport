# AI-Driven Decision Support System for Auckland Public Transport Delay Prediction

This capstone project develops a prototype decision-support system using Auckland Transport GTFS Realtime data and NIWA weather data.

Current milestone:

- Layer 1: Data Sources
- Layer 2: Data Integration and Preprocessing

Initial objective:
Collect sample GTFS Realtime data, obtain NIWA historical weather data, perform sample EDA, and create an integrated dataset for modelling.
